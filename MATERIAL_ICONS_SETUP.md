# Material Design Icons

Material Symbols Rounded is installed in `index.html` through Google Fonts:

```html
<span class="md-icon">calendar_month</span>
```

Use the same helper class anywhere in the static site:

```html
<button>
  <span class="md-icon" aria-hidden="true">photo_camera</span>
  Race photos
</button>
```

For a future npm/React build, install the package version instead:

```bash
npm install material-symbols
```
